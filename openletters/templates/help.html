Get Involved

The Open Correspondence project is in early stages.

We are gradually getting more authors into the system, writing more code and finding the letter data sets. But we could always do with some help or different perspectives...

For example, we are always interested in new data sets if you know of, or have, any that we can use under the Open Licence.

If there is something that you feel is missing, feel free to download the code from our Mecurial repository and experiment or come up with you own service.

The project can be contacted via the Open Literature mailing list.
